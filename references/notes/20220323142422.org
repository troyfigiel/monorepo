*Formula*
\[
C_n = \frac{1}{n+1} \binom{2n}{n} \quad \text{for } n \geq 0
\]

*Recurrence relation*
\[
C_0 = 1 \quad \text{and} \quad C_{n+1} = \sum\limits_{i=0}^n C_i C_{n-i} \quad \text{for } n \geq 0
\]
