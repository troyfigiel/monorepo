#+title: Batch layer
#+filetags: :stub:

The batch layer is a part of the [[id:ec74a43b-86f4-4b01-8201-e0dade746d90][Lambda architecture]].

When we are receiving new data from [[id:8f45a23a-3f0b-4125-8701-fda7ba356b52][Data stream]], the batch layer records it in a [[id:b7eb6abb-47a1-4ae3-a316-3019d1371c87][Master dataset]].
Once we have enough data, we retrain, build and test any downstream [[id:87fb8d8c-c174-4a52-ab30-b92717f3fa54][Machine learning models]].
The new model is then deployed in the [[id:3b676062-8ab9-40ee-964e-d1da4b8a96c0][Speed layer]].
