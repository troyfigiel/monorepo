Text editors like Vim and Emacs will always be around when we need to do administration from the
command line and we need to quickly pop in and out of files, for example when a desktop environment
broke or if we are changing configurations through [[id:090ce4a9-f17c-4b6a-aa3d-1ab538b281f6][SSH]].
