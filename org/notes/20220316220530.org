#+title: Grammar of graphics
#+filetags: :stub:

Building blocks of visualization:
1. Data
2. Transformations
3. Marks
4. Encoding (mapping from fields to mark properties)
5. Scale (functions that map data to visual scales)
6. Guides (axes, legends, etc.)

Visualization concepts should map directly to implementation. Good implementation can influence
good conceptualization. This is extremely powerful for [[id:54c23a92-d7f6-46e0-996e-0d1fd7affc21][Exploratory data analysis]].
