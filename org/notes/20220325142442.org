#+title: Vapnik-Chervonenkis dimension
#+filetags: :stub:

For a set of indicator functions. Let \( \{ \theta(f(x)) \} \), then VC dimension of the set of
indicator functions is equal to h if h is the maximal number of vectors \( x_1, ..., x_h \) that
can be shattered (see [[id:3486294f-3dc9-420d-8c7f-ade2773fcc26][Shattered set]]) using indicator functions from \( \{ \theta(f(x)) \} \). If such vectors
exist for any number h, the VC dimension of the set is infinite.
