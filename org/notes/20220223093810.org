#+title: censored regression model
#+filetags: :stub:

A censored regression model is a model in which the dependent variable is censored above or below a
certain threshold. A standard example is the [[id:aa5722ea-e9f2-4cd0-8fed-c740d7cf4d2e][tobit model]].
