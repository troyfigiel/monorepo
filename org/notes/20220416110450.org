#+title: Hamilton's characteristic function
#+filetags: :stub:

Hamilton's characteristic function \( W(\mathbf{q}) \) is defined in terms of the [[id:e54f8d35-b5e0-4fe3-b3a4-f259d63cbc18][Action]] \(
S(\mathbf{q}, t) = W(\mathbf{q}) - Et \).

If we use Hamilton's characteristic function as a generating function for a [[id:02f213c2-c8c2-4d4f-9551-b956100cb09e][Canonical
transformation]], we end up with the [[id:522261cb-2182-4c15-b8d7-e8e6aea4b3a8][Action-angle coordinates]].
