#+title: Product backlog
#+filetags: :stub:

The product backlog is where a [[id:7027bb9d-cacb-49b4-ba35-ccd38058e548][Product owner]] creates a list of product features using [[id:2b9aa9df-4a48-4611-b947-a3bb75d2ffaa][User stories]].
The [[id:7027bb9d-cacb-49b4-ba35-ccd38058e548][Product owner]] is also responsible for prioritizing the list so the top priorities can be communicated to the team.
