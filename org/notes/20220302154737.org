#+title: Phi coefficient
#+filetags: :stub:

The phi coefficient is a [[id:e444c32f-a2d4-4ac3-9f21-586c54337b32][Correlation coefficient]] used to measure association between two binary variables.

# TODO: Is this true?
When restricted to binary variables, [[id:52d787ad-4e08-4963-a334-7ab8b5d93c87][Pearson correlation coefficient]], [[id:c0c2ddf9-6fc0-4571-9578-cdb201930cec][Kendall rank correlation
coefficient]] and [[id:b53cc863-348d-432c-bd72-e93c740c84ed][Spearman's rank correlation coefficient]] all agree with the phi coefficient.

The phi coefficient can be calculated from a [[id:d458f0f7-ac5e-48a8-8f99-b0d9562b5069][Confusion matrix]] as

\[
\phi = \frac{TP \times TN - FP \times FN}{\sqrt{(TP + FP)(TP + FN)(TN + FP)(TN + FN)}}
\]
