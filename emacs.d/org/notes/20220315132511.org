#+title: Files in the GNU store can only be changed by the Guix build daemon
#+filetags: :stub:

Moreover, you cannot edit any file in the store directly (which includes configurations from [[id:462ccc41-f570-447a-904a-f487e3b7a228][Guix
Home]] or [[id:b2430170-98ab-4801-ad2e-84a1ed686865][Guix System]]), because the store is mounted as a read-only filesystem that only the [[id:17eb2ad4-82ef-42e9-a2a9-11af88f0313e][Guix
build daemon]] can change.
