#+title: The point mass distribution is the only distribution with vanishing entropy
#+filetags: :stub:

\[
S = - \sum\limits_i p_i \log p_i = 0
\]

Then for each state \( p_i \log p_i = 0 \) , which can only happen if \( p_i = 0 \) or \( p_i = 1
\).

Since

\[
\sum\limits_i p_i = 1
\]

we must have the [[id:6bd012d7-b585-48d7-8096-a28770cd7fcf][Point mass distribution]].
