#+title: Logistic regression is a discrete choice model
#+filetags: :stub:

Logistic regression can be derived as a [[id:44af57c9-fb8d-4e57-b8e7-b0a20463e10b][Discrete choice model]] where the error terms follow a [[id:2b10331d-e8d6-46bb-a890-dd7b1892d469][Gumbel
distribution]].

Indeed, if

\[
U_i = \beta_i X + \epsilon_i
\]

then

\[
\mathbb{P}(U_1 > U_0 | X = x) = \mathbb{P}((\beta_1 - \beta_0) x > \epsilon_0 - \epsilon_1)
\]

Since \( \epsilon_i \sim \mathrm{Gumbel}(\mu_i, \alpha) \), we have \( \Delta = \epsilon_0 - \epsilon_1 \sim
\mathrm{Logistic}(\mu_0 - \mu_1,  \alpha) \).

We simplify and let \( \beta_0 = 0 \) and \( \beta_1 = 1 \) as well as let \( \mu_0 = \mu_1 \)
and \( \alpha = 1 \), i.e. if we use the [[id:decc6032-ad49-44aa-a200-320553a6b064][Standard Gumbel distribution]], we find the [[id:76de247c-a9ac-4e8b-a31c-e0e30f519ea6][Standard
logistic distribution]].

Then we end up with

\[
\mathbb{P}(U_1 > U_0 | X = x) = \mathbb{P}(\Delta < x) = \frac{e^x}{1 + e^x}
\]

which is the [[id:041ae89d-f312-4fcc-95d5-c4b9277a0be8][Standard logistic function]]. If we had used the [[id:2b10331d-e8d6-46bb-a890-dd7b1892d469][Gumbel distribution]] instead, we would
end up with the [[id:f4c1dbf3-0f05-4a68-ba2e-b9f6602789d5][Logistic function]].
